# MorseGo
---

A simple morse code parser made in Go.

Features:
- Parse from morse code to ascii or from ascii to morse code;
- It can play the morse code as audio.
- A file can be passed with the --fileName flag


## Install & Build:

Install: 

```bash
	go install
```

Build: 

```bash
	go build
```

## Run:

To see the usage:

```bash
	./morseParser -h
```
