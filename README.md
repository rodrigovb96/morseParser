# MorseGo
---

A simple morse code parser made in Go.

this branch is dedicated to development of the Web version. For CLI go to the master branch.

Install: 

```bash
	go install	
```

Set the PORT:

```bash
	set PORT=8080 # for example	
```

Start the server:

```bash
	./morseParser
```


Then visit:
	localhost:8080


Live Example:
	[Heroku Hosted](still-river-16502.herokuapp.com)
